TrubbleStack
===========

TrubbleStack (Trubble for short) is a modular, open-source, security & compliance
auditing framework which is built in python, using SaltStack as a library.
Trubble provides on-demand profile-based auditing, real-time security event
notifications, alerting and reporting. It also reports the security information
to Splunk, Logstash, or other endpoints. TrubbleStack is a free and open source
project made possible by Adobe.
